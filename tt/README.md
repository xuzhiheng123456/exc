# 简述 #

项目简单实现了一个webserver，包括分发系统，缓存系统，Session系统，模板系统。目前还没有实现form和orm系统。

项目没有使用任何第三方库，纯粹作为http原理教学用。和py-web-server属于同一用途。但是由于形成年代教前者晚，因此代码质量相对较高。而且和前者不同的是，其中没有包含greenlet和非阻塞编程框架，是基于纯粹的线程模型的。
